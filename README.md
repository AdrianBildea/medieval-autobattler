# Medieval Fantasy Autobattler

A Python prototype for a mobile medieval fantasy autobattler game.

## About
This prototype implements the core game logic from the GDD (Game Design Document).
It covers Phase 1 of the development roadmap: **Combat & Basic Units**.

## Features
- ✅ 4 unit classes: Knight, Archer, Fire Mage, Priest
- ✅ Faction synergies: Human, Elf, Orc, Undead
- ✅ Auto-combat loop with AI per class
- ✅ Abilities: Fireball (AoE), Shield Slam (stun), Power Shot, Heal
- ✅ Unit upgrading (3 copies → Level up)
- ✅ Shop system with gold economy
- ✅ Campaign mode (4 levels)

## Tech Stack
- **Language:** Python 3.10+
- **Engine (planned):** Unity or Godot
- **Platform (planned):** Android & iOS

## Run
```bash
python main.py
```
